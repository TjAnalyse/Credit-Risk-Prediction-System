# Credit Risk Prediction System

## Overview

This project predicts the credit risk of loan applicants using Machine Learning.

The system classifies applicants into:

- Low Risk
- Moderate Risk
- High Risk

based on the probability of loan default.

## Model

- Logistic Regression

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Features

- Credit risk prediction
- Risk probability estimation
- Applicant summary
- Recommendation generation

## Dataset

Approximately 982,000 loan records were used to train the model.
